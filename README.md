# Open AutoClicker

Open AutoClicker is a free, open-source Windows auto clicker with a clean black-and-white desktop GUI, bindable mouse and keyboard inputs, configurable toggle hotkey, 1-60 minute run duration, millisecond click interval control, and fully local PowerShell/WinForms behavior with no networking, telemetry, analytics, automatic update checks, external packages, personal data collection, or machine-specific configuration; to use it on Windows, run `Start-AutoClicker.bat`.
